Crowdtimestory

A crowd-powered picture book to audio book converter

View project on GitHub

This project is part of ECE695 offered in Spring '15 at Purdue, where @MensEtManus, @yauc and I created a web application which allows users to create audio books for their favorite picture books through crowd work. This is made possible through HITs(Human Intelligence Task) on Amazon Mechanical Turk.

It is a solid example of utilizing crowd sourcing to produce creative solutions to some problems that computer algorithms cannot solve, yet! During the process of development, we have enhanced our web development skills using Python with Flask framework and JavaScript etc. However, there are endless features that could be implemented for our application on top of the existing features. We hope you enjoy the crowd time stories you create through our web application.
